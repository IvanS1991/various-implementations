# Project Title

Various implementations

### Prerequisites

All you need is Node.js and npm.

### Installing

To install the packages needed:

```
npm install
```

To run the dev script (just a simple JSHint): 

```
npm run dev
```

## Running the tests

To run the tests:

```
npm test
```
